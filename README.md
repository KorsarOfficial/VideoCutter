# Видео Обрезатор

Приложение для обрезки и конвертации видео в различные форматы.

## Функциональность

- Обрезка фрагментов из видео
- Конвертация в различные форматы: MP4, AVI, MOV, MKV, WebM, OGG, FLV, WMV
- Поддержка Drag-and-Drop для загрузки видео
- Удобный графический интерфейс

## Требования

- Python 3.8 или выше
- FFmpeg (должен быть установлен и доступен в PATH)

## Установка

1. Клонируйте репозиторий:
```
git clone <url-репозитория>
cd VideoCutter
```

2. Установите необходимые зависимости:
```
pip install -r requirements.txt
```

3. Установите FFmpeg:
   - **Windows**: Скачайте с [официального сайта](https://ffmpeg.org/download.html) и добавьте в PATH
   - **Mac OS**: `brew install ffmpeg`
   - **Linux**: `sudo apt install ffmpeg` или аналогичная команда для вашего дистрибутива

## Запуск приложения

```
python main.py
```

## Использование

1. Запустите приложение
2. Нажмите на кнопку "+" или перетащите видеофайл в область приложения
3. Выберите начало и конец фрагмента с помощью слайдеров
4. Выберите формат выходного файла
5. Нажмите "Обрезать"
6. Выберите расположение для сохранения обрезанного видео
